# 可拖拽Table

继承自 Ant.Design Table 组件  
提供表头可拖拽功能，提供自定义客制化表头能力，增强业务表格适应性

## 何时使用

提供一个列数据可拖拽表格，表格列排序数据存储在浏览器端。 需要批量展示数据时，且对数据有操作、过滤、排序需求时使用。
